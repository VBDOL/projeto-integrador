# Projeto Integrador - Refatorado

Este repositório contém duas pastas: `backend` (API Node.js + SQLite) e `frontend` (React).

Instruções rápidas:

1. Backend
   - cd backend
   - npm install
   - npm run dev

2. Frontend
   - cd frontend
   - npm install
   - npm start

Deploy sugerido:
- Backend: Render, Railway ou Heroku
- Frontend: Netlify ou Vercel
